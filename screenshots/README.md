# Custom Screenshots Folder

This directory is reserve compiled for storing the beautiful, high-fidelity responsive captures of the **Auren Luxury Travel** digital interface.

### 📸 How to populate your screenshots:
1. Save your 10 captured interface screenshots into this exact directory.
2. Name them sequentially as:
   - `screenshot_01.png` (Landing page / Hero)
   - `screenshot_02.png` (Alba di Amalfi detail / Curated highlights)
   - `screenshot_03.png` (Sanctuaries navigation slider)
   - `screenshot_04.png` (Curated grand itineraries & booking engine)
   - `screenshot_05.png` (Additional privileges, flights & custom extras)
   - `screenshot_06.png` (Sovereign Telemetries statistics)
   - `screenshot_07.png` (Timeland Sanctuary philosophy quote)
   - `screenshot_08.png` (Patron circle reviews)
   - `screenshot_09.png` (The Sovereign Journal - CMS)
   - `screenshot_10.png` (Secure Bespoke Concierge Desk Form)

When you export this repository to GitHub, your screenshots will automatically load into the magnificent gallery section implemented at the bottom of the main `README.md` file!
